Notre programme complet se compose de 18 classes.
Celles-ci sont decrites dans notre rapport de projet.
Notre fenetre Main est la classe appelee Tetris.
Pour lancer le jeu, il faut alors compiler chaque classe deux fois (pour être sûr que tout compile bien), puis compiler et executer la classe Tetris.
N'hésitez pas à lire les règles dans le menu pour voir comment fonctionne le jeu et quelles sont les touches utiles pour jouer.
